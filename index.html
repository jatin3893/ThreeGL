Version wise progress of this project

  1. Voxel Builder v1.0

    • Mouse controls
    • Left click: Create a Voxel
    • Shift + Left Click: Delete a voxel
    • Click + Drag: Rotate around the center
    • Left + Right Click and Drag: Pan')
    • Scroll: Zoom
    • Keys 0-9: Different coloured Voxels
    • It is still a work in progress! This project takes inspiration from the examples provided by Three.js
    • Made using Three.js: http://threejs.org
    • Read more about voxels on wikipedia: https://en.wikipedia.org/wiki/Voxel
